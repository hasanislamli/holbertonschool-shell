# Init Files, Variables and Expansions

This project focuses on Bash shell initialization files, variables, aliases, and expansions.

It covers:
- Creating and using aliases
- Local and global variables
- Environment variables (HOME, PATH, PS1)
- Shell expansions and command substitution
- Arithmetic operations in the shell
- Using commands like printenv, set, unset, export, alias, unalias, source, printf

Each script demonstrates a specific concept related to shell behavior and configuration.

All scripts:
- Are written in Bash
- Contain exactly 2 lines
- Start with #!/bin/bash
- Are executable
