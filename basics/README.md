# Shell Basics

This project contains basic shell scripts written for learning purposes.

Each script performs a simple task related to navigation, file manipulation, and shell commands.

## Files

- 0-current_working_directory: prints the absolute path of the current directory
- 1-listit: lists directory contents
- 2-bring_me_home: changes directory to home
- 3-listfiles: displays current directory contents in long format
- 4-listmorefiles: displays hidden files
- 5-listfilesdigitonly: displays files with numeric IDs
- 6-firstdirectory: creates a directory
- 7-movethatfile: moves a file
- 8-firstdelete: deletes a file
- 9-firstdirdeletion: deletes a directory
- 10-back: moves to previous directory
- 11-lists: lists files in multiple directories
- 12-file_type: prints file type
- 13-symbolic_link: creates symbolic link
- 14-copy_html: copies HTML files
- 15-lets_move: moves files
- 16-clean_emacs: deletes Emacs backup files
- 17-tree: creates a directory tree structure
