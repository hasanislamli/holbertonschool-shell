# Shell Permissions

This project contains basic shell scripts related to file permissions and user management in Linux.

## Scripts

- 0-iam_betty: switches the current user to betty
- 1-who_am_i: prints the effective username of the current user
- 2-groups: prints all the groups the current user belongs to
- 3-new_owner: changes the owner of the file hello to betty
- 4-empty: creates an empty file called hello
- 5-execute: adds execute permission to the owner of the file hello
- 6-multiple_permissions: adds execute permission to owner and group, read permission to others
- 7-everybody: adds execution permission to owner, group and others
- 8-James_Bond: sets permission so that only others have all permissions
- 9-John_Doe: sets the mode of the file hello to -rwxr-x-wx
- 10-mirror_permissions: sets permissions of file hello same as olleh
- 11-directories_permissions: adds execute permission to all subdirectories
- 12-directory_permissions: creates a directory with specific permissions
- 13-change_group: changes the group owner to school
- 14-change_owner_and_group: changes owner to vincent and group to staff
- 15-symbolic_link_permissions: changes owner and group of a symbolic link
- 16-if_only: changes owner only if it is owned by guillaume
